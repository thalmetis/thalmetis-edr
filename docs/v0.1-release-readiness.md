# v0.1 Release Readiness

This is an internal release-readiness checklist for `thalmetis-edr` v0.1. It
tracks release preparation and post-release follow-up without itself executing
publication, tagging, archiving, or PyPI release steps.

## Scope

v0.1 release readiness covers the bounded McRae 2024 Figure 5a / Table 3
pathway reconstruction, packaged source data, validation metadata, example
script, notebook, documentation, and CI checks.

The release candidate remains research and educational reproducibility work. It
is not CFD, not a validated industrial viability predictor, not GMP or
process-control software, and not a process decision tool.

## Validation Status

Before release approval, confirm:

- local editable install smoke check passes
- pytest passes
- Ruff passes
- example smoke run passes
- package import and `reproduce_table3()` validation-summary smoke check passes
- notebook code-cell smoke check passes as local validation practice
- GitHub Actions CI is green for the release candidate before approval

## Release-Readiness Checklist

- local `main` is clean and synced with `origin/main`
- CI is green before release approval
- README, docs, example script, and notebook are present
- package install smoke check is complete
- before release, confirm no `v0.1.0` tag already exists
- before release, confirm no GitHub release already exists for `v0.1.0`
- before release, confirm no PyPI publication has been performed unless explicitly approved
- before release, confirm no Zenodo DOI has already been recorded
- after release, record the GitHub release URL and Zenodo DOI if created

## Post-Approval Release Planning

After explicit release approval, plan these steps separately:

- tag planning only
- GitHub release planning only
- Zenodo planning only

No tag, GitHub release, Zenodo DOI, or publishing action is executed by this
documentation metadata PR.

## Explicit Non-Actions for This PR

- no tag created by this documentation metadata PR
- no GitHub release created by this documentation metadata PR
- no PyPI publishing performed by this documentation metadata PR
- no Zenodo DOI created by this documentation metadata PR
- no scientific or model scope changed
