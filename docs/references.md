# References

The primary scientific sources for the v0.1 release materials are:

- McRae et al. 2024. Primary source for the v0.1 McRae 2024 pinch-off /
  sparging model and Table 3 reproduction.
- Walls et al. 2017. Source for affected-volume framing and the critique of
  relying on a single maximum EDR; v0.1 treats Walls et al. 2017 as
  affected-volume-only context.

Users should cite the package and should also cite McRae et al. 2024 and Walls
et al. 2017 when using the relevant scientific framing.
