# References

The primary scientific sources for the package and v0.2 release materials are:

- McRae et al. 2024. Primary source for the pinch-off Figure 5a / Table 3 /
  Equation 3 pathway. DOI: `10.1002/bit.28582`.
- Walls et al. 2017. Source for affected-volume framing and the critique of
  relying on a single maximum EDR. DOI: `10.1038/s41598-017-14531-5`.

Users should cite the package and should also cite the relevant scientific
papers when using their scientific framing, data, equations, or interpretation.
